    Update call graph

    Use "Call graph" -> "Update..." or "Update" button on tool bar or "Ctrl+U".



    1. Select call graph for updating.

    2. Add call graphs that were changed to the list, use "Add..." and "Delete" buttons for items managing.

    3. Select full name of output call graph.

    Press the "OK" button to update call graph.
